Getting Started

To customize this framework, you can:

  1. Define your routes in routes/web.php
  2. Create controllers in app/Controllers/
  3. Create models in app/Models/
  4. Create views in templates/
  5. Configure the application in config/app.php

Example Route

$router->get('/hello/{name}', function ($name) {
    return view('hello', ['name' => $name]);
});

Example Controller Method

public function show($id)
{
    $item = $this->itemService->find($id);
    return $this->view('items/show', ['item' => $item]);
}
